# Concierge Management Center

Eigenständige interne Seite für Alpine Concierge Tirol.

- Startdatei: `index.html`
- Passwort: `ACT2026`
- Speicherung: lokal im Browser per `localStorage`
- Backup: JSON-Export und JSON-Import im Bereich `Backup`
- Druck/PDF: über die Buttons in `Angebot` und `Programm`

Die öffentliche Alpine-Concierge-Tirol-Webseite liegt nicht in diesem Projekt und wurde nicht verändert.
